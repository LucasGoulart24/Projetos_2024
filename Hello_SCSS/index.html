O que é SASS?

É uma linguagem de extensão do CSS que permite escrever código de forma mais organizada e intuitiva. A sigla Sass significa "Syntactically Awesome Style Sheets", que traduzido para o português significa "Folhas de Estilo com uma Sintaxe Incrível".
